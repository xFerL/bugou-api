# Bugou API

API REST para tradução e catalogação de erros técnicos.

## Objetivo

Transformar mensagens de erro técnicas em explicações claras, causas prováveis e possíveis soluções.

## Tecnologias

* Java 25
* Spring Boot
* Spring Data JPA
* H2 Database
* Lombok
* Validation
* Maven
